Setting up Documentation:

  • OSX:
  1. Install doxygen: using [Homebrew](/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"): 'brew install doxygen'
  2. Install nodejs: 'brew install node'
  3. cd into this repository
  4. run 'npm install' (this installs all the required node modules I've used in this project)
